# 🌐 Personal Portfolio Website
A responsive portfolio website built using **HTML, CSS, and Bootstrap** to showcase my education, technical skills, projects, and experience. This project was developed as part of my Full Stack Web Development journey and helped strengthen my frontend development skills.

# Features
- Responsive design for desktop and mobile devices
- Clean and modern user interface
- About section introducing my background and interests
- Resume section displaying education, technical skills, and academic projects
- Web Development Projects section showcasing projects completed during my learning journey
- Experience section highlighting hackathons, competitions, and startup experience
- Easy navigation using a responsive navigation bar

# What I Learned
While building this project, I gained practical experience in:
- Building responsive web pages using Bootstrap
- Creating reusable layouts with Bootstrap components
- Using Flexbox and Grid for page layouts
- Implementing media queries for responsiveness
- Organizing HTML and CSS files for better maintainability
- Designing a clean and user-friendly interface

# Technologies Used
- HTML5
- CSS3
- Bootstrap 5
